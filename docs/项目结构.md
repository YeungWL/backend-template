#### 项目结构

|               目录 | 作用               |
| -----------------: | :----------------- |
|              /docs | 项目相关的文档     |
|            /public | 静态页面容器       |
|               /src | 核心代码           |
|        /src/assets | 静态资源文件       |
|  /src/assets/icons | 图标文件           |
| /src/assets/images | 图片文件           |
|    /src/components | 自定义组件         |
|       /src/layouts | 页面视图容器       |
|       /src/plugins | 全局插件和扩展功能 |
|        /src/router | 路由               |
|         /src/store | vuex               |
|         /src/utils | 自定义工具和方法   |
|         /src/views | 页面文件           |
